# Day 3 - Python Dictionary Practice (Part 1)

This file contains three beginner-level problems using **Python dictionaries**.  
Instead of showing the full code, here’s an explanation of how each program works.

---

## 📘 Problem 1: Country Capitals
- A dictionary is created with country names as **keys** and their capitals as **values**.
- The program prints the dictionary so the user can see available options.
- The user is asked to type a country name.
- If the country exists in the dictionary, the program prints its capital.
- If not, it shows a message saying the country is not found.

---

## 📘 Problem 2: Student Marks Average
- A dictionary stores student names as **keys** and their marks as **values**.
- The program calculates the **sum of all marks** using `sum()`.
- It divides the total by the number of students (`len()`) to get the average.
- Finally, it prints the average marks of all students.

---

## 📘 Problem 3: User Age Lookup
- A dictionary is created with names as **keys** and ages as **values**.
- The user is asked to enter a name.
- If the name exists in the dictionary, the program prints the age.
- If the name is not found, it prints `"User not found"`.

---

## 📝 Notes
- These problems practice **dictionary lookups**, **value operations**, and **user input handling**.
- Problem 1 → Key lookup with countries.
- Problem 2 → Working with values (`sum`, `len`).
- Problem 3 → Conditional lookup with error handling.

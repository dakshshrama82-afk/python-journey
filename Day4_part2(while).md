# Day 4 - Part 2 🚀

This section covers loop practice problems in Python using **while loops**.

---

## 📌 Question 1  
**Print numbers from 1 to 100**  
- Start with `i = 1`.  
- Use a `while` loop that runs until `i <= 100`.  
- Print each number and increment `i` by 1.  
- After completion, print `"end of loop"`.  

---

## 📌 Question 2  
**Print numbers from 100 to 1**  
- Start with `i = 100`.  
- Use a `while` loop that runs until `i >= 1`.  
- Print each number and decrement `i` by 1.  
- After completion, print `"end of loop"`.  

---

## 📌 Question 3  
**Print the multiplication table of a number `n`**  
- Start with `i = 1`.  
- Use a `while` loop that runs until `i <= 10`.  
- Print `n × i` for each iteration.  
- Increment `i` by 1 each time.  
- After completion, print `"end of loop"`.  

---

## 📌 Question 4  
**Print the elements of a list using a loop**  
- A list of squares is defined.  
- Start with `i = 1`.  
- Use a `while` loop that runs until `i <= 10`.  
- Print `i ** 2` for each iteration.  
- Increment `i` by 1 each time.  
- After completion, print `"end of loop"`.  

---

## 📌 Question 5  
**Search for a number `x` in a tuple using a loop**  
- A tuple of squares is defined.  
- The user enters a number to search.  
- Start with `i = 0`.  
- Use a `while` loop that runs until `i < len(tuple)`.  
- If the number matches, print the index and break.  
- If not, print `"not found"` for that index.  
- After completion, print `"end of loop"`.

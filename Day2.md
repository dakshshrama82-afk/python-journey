## 📘 Day 2 Overview
<br>
- **Problem 1: Traffic Light Simulator**  
<br>
  Ask the user for the current traffic light color (red, yellow, green) and print the correct driving action.
  <br>
  **Concepts used:** `if`, `elif`, `else` statements, string comparison
  <br>
- **Problem 2: Café Menu Order**
<br>
  Ask the user to choose an item (pizza, burger, coffee) and print its price.  
  <br>
  **Concepts used:** Dictionaries, conditional branching, user input handling
  <br>
- **Problem 3: Café Order with Quantity**  
<br>
  Ask the user to choose an item and enter quantity, then calculate and print the total bill.  
  <br>
  **Concepts used:** Dictionaries, arithmetic operations, conditionals, type conversion

## 📘 Problem 1: Traffic Light Simulator
<br>
- The program asks the user to enter the current traffic light color (red, yellow, or green).<br>
- If the color is `"red"`, it prints `"Stop. Wait until green."`<br>
- If the color is `"yellow"`, it prints `"Slow down."`<br>
- If the color is `"green"`, it prints `"Go."`<br>
- If the input doesn’t match any of these, it prints `"Light is broken."`<br>

<br>

## 📘 Problem 2: Café Menu Order
<br>
- A dictionary stores menu items (`pizza`, `burger`, `coffee`) with their prices.<br>
- The program prints the menu and asks the user to choose an item.<br>
- If the item exists, it prints the price.<br>
- If the item doesn’t exist, it prints `"Item not available."`<br>
- This problem practices dictionary lookups and conditional branching.<br>

---
<br>

## 📘 Problem 3: Café Order with Quantity
<br>
- A dictionary stores menu items with their prices as integers.<br>
- The program asks the user to choose an item and enter a quantity.<br>
- It calculates the total bill by multiplying the price with the quantity.<br>
- If the item doesn’t exist, it prints `"Item not available."`<br>
- This problem practices dictionary lookups, arithmetic operations, and type conversion.<br>

# Day 4 - Part 1 🚀

This section covers loop practice problems in Python using **for loops**.

---

## 📌 Question 1  
**Print the elements of the list using a loop**  
- A list of numbers is created.  
- The program uses a `for` loop to print each element.  
- An `else` statement is used to print `"END OF LOOP"` once the loop finishes.  

---

## 📌 Question 2  
**Search for a number `x` in the list using a loop**  
- A list of numbers is defined.  
- The user enters a number to search.  
- A `for` loop checks each element with its index.  
- If found, it prints the element and index, then breaks the loop.  
- If not found, the `else` block prints `"do not exist"`.  

---

## 📌 Question 3  
**Print numbers from 1 to 100**  
- A `for` loop with `range(1,101)` prints numbers from 1 to 100.  
- After completion, `"end of loop"` is printed.  

---

## 📌 Question 4  
**Print numbers from 100 to 1**  
- A `for` loop with `range(100,0,-1)` prints numbers in reverse order.  
- After completion, `"end of loop"` is printed.  

---

## 📌 Question 5  
**Print the multiplication table of a number `n`**  
- A `for` loop with `range()` generates multiples of a number.  
- Each multiple is printed, followed by `"end of loop"`.

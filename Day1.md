<br>

## 📘 Problem 1: Greeting with Input
<br>
- The program asks the user for their **name** and **age**.
<br>
- It then prints a personalized greeting in the format:
<br>
  `"Hello <name>, you are <age> years old!"`
<br>
- This helps practice `print()` and `input()` functions along with type conversion (`int`).
<br>

---
<br>

## 📘 Problem 2: Simple Calculator
<br>
- The program asks the user to enter **two numbers**.
<br>
- It calculates and prints their **sum, difference, product, and quotient**.
<br>
- Division by zero is handled with an `if` condition to avoid errors.
<br>
- This problem practices arithmetic operations and conditional statements.
<br>

---
<br>

## 📘 Problem 3: Even or Odd
<br>
- The program asks the user to enter a number.
<br>
- It checks whether the number is **even or odd** using the modulus operator (`%`).
<br>
- If divisible by 2, it prints `"Even"`, otherwise `"Odd"`.
<br>
- This problem reinforces conditional logic and mathematical operators.
<br>

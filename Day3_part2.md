# Day 3 - Python Set Practice (Part 2)

This file contains three beginner-level problems using **Python sets**.
Each problem demonstrates how sets handle uniqueness, operations, and membership tests.

---

## 📘 Problem 1: Unique Elements
- The program asks the user to enter **eight numbers** one by one.
- These numbers are stored inside a **set**.
- Since sets automatically remove duplicates, the final output only shows **unique numbers**.
- Example: If you enter `10, 20, 20, 30`, the output will be `{10, 20, 30}`.


---


## 📘 Problem 2: Set Operations
- Two sets are created:
  - `A = {1, 2, 3, 4}`
  - `B = {3, 4, 5, 6}`
- The program performs different set operations:
  - **Union** → Combines all elements from both sets.
  - **Intersection** → Shows common elements between the sets.
  - **Difference (A − B)** → Shows elements in A but not in B.
  - **Difference (B − A)** → Shows elements in B but not in A.
- This demonstrates how sets can be used for mathematical-style operations.

---

## 📘 Problem 3: Membership Test
- A set of colors is created: `{"red", "green", "black"}`.
- The user is asked to enter a color name.
- The program checks if the entered color exists in the set.
- If found, it prints `"Yes, present in the set"`.
- If not found, it prints `"Do, not present"` (meaning the color is missing).


---


## 📝 Notes
- These problems practice **set basics**: uniqueness, operations, and membership.
- Problem 1 → Removing duplicates.
- Problem 2 → Performing union, intersection, and difference.
- Problem 3 → Checking if an element exists in a set.
